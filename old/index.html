Variables

Variables are containers for storing data (storing data values).

There are 4 Ways to Declare a JavaScript Variable:

1. let

2. const

3. var

4. Nothing

1. Variables defined with let cannot be Redeclared.

1. Variables defined with let must be Declared before use.

1. Variables defined with let have Block Scope.

2. Variables defined with const cannot be Redeclared.

2. Variables defined with const cannot be Reassigned.

2. Variables defined with const have Block Scope.

Let Example

let x = 5; let y = 6; let z = x + y;

var Example

var x = 5; var y = 6; var z = x + y;

Const Example

const PI = 3.141592653589793;
